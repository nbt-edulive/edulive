### Bài 9.15 trang 96 SGK Toán 11 Kết nối
Cho hàm số P\(x\) = ax2 \+ bx + 3\(a, b là hằng số\). Tìm a, b biết P′\(1\) = 0 và P"\(1\) = −2.
**Bài làm**
Ta có P′\(1\) = 0
P′\(1\) = 2a\(1\) + b = 0 ⇒ 2a = −b
Với P"\(1\) = −2
P"\(1\) = 2a\(−1\) = −2 ⇒ a = 1
Vậy a = 1 và b = −2, a = −2
⇒ P\(x\) = x2 − 2x + 3
