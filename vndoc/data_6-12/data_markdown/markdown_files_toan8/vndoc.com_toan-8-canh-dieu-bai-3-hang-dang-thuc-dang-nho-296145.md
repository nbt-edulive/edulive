## I. Hằng đẳng thức
**Hoạt động 1 trang 18 sgk Toán 8 tập 1 CD** : Xét hai biểu thức: P = 2\(x+y\) và Q = 2x+2y. Tính giá trị của mỗi biểu thức P và Q rồi so sánh hai giá trị đó trong mỗi trường hợp sau:
a. Tại x = 1; y = - 1.
b. Tại x = 2; y = - 3
Bài giải
a. Tại x = 1; y = - 1
P = 2\(1-1\) = 0
Q = 2.1-2.1 = 0
=> P = Q
b. Tại x = 2; y = - 3
P = 2\(2-3\) = -2
Q = 2.2+2.\(-3\) = -2
=> P = Q
