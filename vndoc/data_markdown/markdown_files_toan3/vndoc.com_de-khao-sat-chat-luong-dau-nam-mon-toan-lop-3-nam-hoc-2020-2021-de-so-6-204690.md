---|---|---|---|---  
D| C| B| C| C  
**II. Phần tự luận**
**Bài 1:**
17cm + 24cm = 41cm| 34dm + 16dm = 50dm| 24kg – 17kg = 7kg  
---|---|---  
**Bài 2:**
234 < 267| 123 + 435 > 555| 984 – 362 = 111 + 511  
---|---|---  
**Bài 3:**
Chu vi của hình tứ giác là:
3 + 2 + 4 + 6 = 15 \(cm\)
Đáp số: 15cm
**Bài 4** :
Hoa có tổng cộng số quyển vở là:
18 + 27 = 45 \(quyển vở\)
Đáp số: 45 quyển vở
