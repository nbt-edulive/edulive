## **Câu 1 trang 84 SGK Tiếng Việt lớp 3 Tập 2**
**** Viết tên riêng: Ý Yên
**Trả lời:**
\- Học sinh luyện viết tên riêng: Ý Yên

**** Viết câu:

Hè về, sen tỏa ngát.

\+ Viết hoa chữ cái đầu câu: Xuân, Hè.
