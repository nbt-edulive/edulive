_Bài giải_

……………………………………………………………………………………………………………

……………………………………………………………………………………………………………
**Lời giải**
So sánh: 247 560 > 85 500 nên tàu màu trắng chở nhiều dầu hơn.
Tàu màu trắng chở nhiều hơn tàu màu đỏ số dầu là:
247 560 – 85 500 = 162 060 \(_l_\)
Đáp số: 162 060 lít dầu
